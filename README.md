### Hard Skills  

+ **Python** - pandas, numpy, matplotlib, seaborn  
+ **Jupyther** - создание структурированных аналитических отчетов: код, графики, интерпретация результатов.  
+ **SQL** - ad-hoc, оконные функции, подзапросы, CTE, интеграция с BI-инструментами, временные таблицы, JOIN GROUP BY UNION (PostgreSQL, MySQL)  
+ **Excel** - сводные таблицы, макросы, VLOOKUP, XLOOKUP, MATCH + INDEX, интеграция с внешними источниками данных (CSV, XML)  
+ **DataLens** - создание дашбордов, объединение источников данных, вычисляемые поля, фильтры  
+ Git, Linux
